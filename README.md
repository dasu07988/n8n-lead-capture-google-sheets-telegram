# n8n Lead Capture → Google Sheets → Telegram

## Overview

This is a production-ready n8n workflow template that captures new leads through a Webhook, stores them in Google Sheets, and sends instant Telegram notifications.

## Features

- Webhook Lead Capture
- Google Sheets Integration
- Telegram Notification
- Input Validation
- Production Ready
- Easy Setup

## Tech Stack

- n8n
- Google Sheets
- Telegram Bot API
